# Exercício 1 - Controle de Aluguel de Quadra Esportiva

Tabelas
-Cliente
-Horario
-Aluguel


Regras de Negócio
-Não permitir cadastrar cliente com nome e número vazio
-Não permitir reservar horário já ocupado no mesmo dia
-Registrar aluguel informando cliente, horário e valor
-Calcular automaticamente valor total quando houver mais de um horário no mesmo dia
-Permitir consultar todos os aluguéis de uma determinada data
